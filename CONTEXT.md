# The AEther Flow Website

This context defines the public presentation layer for The AEther Flow research
project and its boundary with the source research repository.

## Language

**The AEther Flow Website**:
The public-facing presentation surface for explaining, describing, and promoting
The AEther Flow project to readers.
_Avoid_: Source repository, research-control repository

**The AEther Flow Project**:
The upstream research project whose claims, documents, governance state, and
validated source materials are presented by the website.
_Avoid_: Website, publication layer

**Source Authority Boundary**:
The rule that scientific, mathematical, governance, and research-workflow claims
must remain grounded in the upstream source project rather than being invented or
silently changed by website copy.
_Avoid_: Content preference, style guideline

**Reader-Facing Material**:
Public explanatory content derived from reviewed or approved project sources for
a non-internal audience.
_Avoid_: Source material, research-control state
