# Source-Authority Repair Example

## Scenario

A proposed public summary overstates the exact-closure result.

## Draft

```text
The exact-closure package proves the Æther substrate and completes the
first-principles derivation of general relativity.
```

## Gate Result

`repair`

The current website claim registry identifies both propositions as forbidden
overreads. The available accepted evidence supports a qualified effective-level
statement and an explicit open-foundational limitation.

## Repaired Text

```text
At the effective level, the active exact-closure package is complete as an
operational theory statement: its observable gravitational content is that of
general relativity. A first-principles derivation of Einsteinian gravity from
explicit Æther or Æther-Flow substrate variables remains open.
```

## Grounding

- The exact wording and qualification must be rechecked against the current
  accepted statement registry before public use.
- Upstream registered TeX sources remain authoritative; the website registry
  is publication-governance evidence, not independent scientific validation.
- A warning-pattern scan may identify inflated language, but it cannot validate
  the repaired physics statement.
