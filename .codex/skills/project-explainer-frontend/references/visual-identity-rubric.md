# Visual Identity Rubric

Use visual references as style evidence, not as sources for project claims.

## Palette Extraction

The identity brief should identify:

- Background color candidate.
- Surface color candidate.
- Primary text color candidate.
- Muted text color candidate.
- Primary accent.
- Secondary accent.
- Border or divider color.

Check contrast before using any color pair in the final UI.

## Typography Inference

Raster images rarely provide reliable exact font identification. Infer style
direction rather than claiming a precise font unless the font is visible or the
user supplied it.

Useful directions for this project include:

- Technical and geometric.
- Editorial and precise.
- Archival or source-document oriented.
- Laboratory-console restrained.

## Distinctiveness Gate

Use one visual signature tied to the project, such as an evidence ledger,
source-boundary panel, publication-map structure, or theorem-status treatment.
Do not stack many unrelated effects.
